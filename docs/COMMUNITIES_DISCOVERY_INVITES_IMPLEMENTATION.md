# Communities Discovery + Invites UX

Этот проход превращает сообщества из ручного списка в более живую систему поиска, рекомендаций, invite-кодов и отображения в профиле.

## Что добавлено

- Подборки сообществ:
  - рекомендации по пересечению тегов и текущему участию пользователя;
  - популярные открытые/закрытые сообщества;
  - похожие сообщества на странице конкретного сообщества.
- Invite UX без email и телефонов:
  - проверка invite-кода;
  - показ сообщества, к которому ведёт код;
  - принятие invite-кода;
  - прямое вступление по валидному коду даже в закрытое/приватное сообщество;
  - защита от просроченных, отозванных, исчерпанных и чужих invite-кодов.
- Сообщества в профиле:
  - блок сообществ в своём профиле;
  - блок сообществ в профиле другого пользователя;
  - приватность блока через настройку `community_visibility`.
- Новая настройка приватности:
  - `Сообщества в профиле`: все / мой круг / друзья / никто.

## Новые API

- `GET /api/communities/discovery`
- `GET /api/communities/[slug]/similar`
- `GET /api/communities/invites?code=...`
- `POST /api/communities/invites`
- `GET /api/users/[id]/communities`

## Изменения базы

В `UserPreference` добавлено поле:

```prisma
communityVisibility String @default("connections")
```

После установки архива выполнить:

```bash
npx prisma generate
npx prisma db push
```

## Без email/phone

Вся логика приглашений остаётся внутри Friendscape:

- нет email;
- нет телефона;
- нет внешней адресной книги;
- invite-код проверяется и принимается только авторизованным пользователем.
