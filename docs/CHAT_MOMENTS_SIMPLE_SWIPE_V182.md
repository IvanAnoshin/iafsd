# v182 — Messenger moments simple swipe

## Что изменено

Механика скрытия моментов в `/chat` упрощена.

Теперь один уверенный свайп вверх по области сайдбара скрывает только блок «Моменты». Фильтры остаются на месте, а список переписок получает больше места и скроллится обычным нативным скроллом.

## Почему так

Предыдущая версия пыталась одновременно скрывать весь блок контролов, подталкивать scrollTop и автоматически возвращать элементы при движении назад. Это могло давать дёрганье на телефоне.

В v182 логика проще:

- свайп вверх скрывает моменты;
- список чатов скроллится сам;
- фильтры не исчезают;
- нет программного scroll-boost во время жеста;
- нет постоянного пересчёта высоты на каждом touchmove.

## Как вернуть моменты

Если список чатов находится в самом верху, свайп вниз возвращает блок моментов.

## Что проверить вручную

1. Открыть `/chat` на мобильной ширине.
2. Сделать один свайп вверх по блоку моментов/фильтров.
3. Блок «Моменты» должен скрыться без дёрганья.
4. Фильтры должны остаться видимыми.
5. Список переписок должен скроллиться обычным образом.
6. Вернуться к самому верху списка и свайпнуть вниз — моменты должны появиться снова.
