# PROFILE COMPOSER SITE ATTACH AUTORESIZE V129

## Что сделано
- Кнопка скрепки больше не открывает системный файловый диалог сразу.
- Вместо этого открывается внутренняя форма выбора вложений в стиле сайта.
- В форме выбора добавлена зона drag-and-drop для фото и видео.
- Системный выбор файла остаётся только как явное действие «Выбрать на устройстве», потому что браузер не даёт сайтам получать локальные файлы без разрешения пользователя.
- Поле текста публикации больше нельзя растягивать вручную.
- Поле публикации автоматически растёт по высоте во время набора текста.
- Максимальная высота поля ограничена, после чего включается внутренний скролл.
