# Communities feed implementation

Этот проход превращает сообщества из карточек членства в живую часть продукта.

## Что добавлено

- `Post.communityId` и связь `Community.posts` в Prisma.
- Поля поста для community scope: `visibility`, `isPinned`.
- API `GET /api/communities/[slug]/posts` — лента постов сообщества.
- API `POST /api/communities/[slug]/posts` — публикация в сообщество.
- Проверка прав публикации через `postingPermission` и роль участника.
- Проверка видимости ленты: публичные сообщества читаются, закрытые/приватные — только участниками.
- Вкладки страницы сообщества: `Лента`, `Правила`, `Участники`, `Заявки`, `Настройки`.
- Реальный composer постов внутри сообщества.
- Пустые состояния без фейковых карточек.
- Подмешивание публичных постов сообществ в общую ленту.
- CSRF-заголовки для действий на страницах сообществ.

## Что осталось следующим проходом

- Редактирование настроек сообщества из UI.
- Редактирование правил.
- Пин/анпин постов сообщества.
- Удаление/скрытие постов модераторами.
- Мут/бан участника.
- Раздел медиа сообщества.
- Жалобы на сообщество и community moderation queue.
