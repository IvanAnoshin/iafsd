# V153 — стабилизация создания момента в мессенджере

Короткий KISS-проход без новой backend-логики.

## Что изменено

- Sheet создания момента открывается свежим состоянием без старой ошибки/черновика.
- При закрытии sheet очищает локальный черновик и ошибку.
- Добавлена защита от двойной публикации через `momentPublishingRef`.
- Добавлена мягкая валидация: пустой момент не отправляется.
- Добавлен счётчик символов `0/180`.
- Ошибки сети/сессии показываются более понятным текстом.
- При `401/403` публикация один раз пробует обновить CSRF-токен и повторить запрос.
- На время открытого sheet блокируется фоновой скролл body.
- Escape закрывает sheet, если публикация не идёт.
- После успешной публикации момент добавляется в рейл без дубликата и черновик очищается.
