# PROFILE COMPOSER MOBILE NO DRAGDROP V130

## Что сделано
- На мобильных и touch-устройствах скрыта drag-and-drop зона в форме прикрепления медиа.
- Drag-and-drop обработчики теперь вообще не навешиваются, если устройство не поддерживает нормальный desktop drag-and-drop сценарий.
- На мобильных остаётся только понятный сценарий выбора файла через кнопку «Выбрать на устройстве».
- Добавлен CSS fallback через media query `(hover: none), (pointer: coarse)`, чтобы зона перетаскивания не отображалась даже при сбое JS-детекта.
