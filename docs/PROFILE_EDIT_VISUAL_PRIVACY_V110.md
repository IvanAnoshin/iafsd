# Profile edit visual and privacy pass v110

Цель прохода: сделать full screen редактор профиля полноценным экраном, а не просто формой с несколькими полями.

## Что добавлено

- Превью профиля с обложкой и аватаром.
- Загрузка аватара и обложки через существующий безопасный upload-пайплайн медиа.
- Отдельный цветовой акцент для обложки.
- Интересы профиля: до 8 коротких тегов.
- Быстрые настройки приватности прямо в редакторе профиля:
  - профиль;
  - фото и видео;
  - активность;
  - сообщества.
- Сохранение профиля и приватности одним действием.

## Хранение

В `UserPublicProfile` добавлены поля:

- `coverTone`
- `avatarUrl`
- `coverUrl`
- `interests`

Для базы добавлена миграция `20260427_profile_visual_interests`.

## UX-принципы

- Редактор остаётся full screen, без нижней навигации.
- Пользователь видит превью до сохранения.
- Приватность вынесена в редактор профиля, но не дублирует весь экран настроек.
- Реализация сохранена простой: один экран, один submit, без тяжёлого роутинга и лишних файлов.
