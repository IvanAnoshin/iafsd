# PROFILE_EDIT_FULLSCREEN_V109

## Что изменено

- Редактирование профиля вынесено из вкладки «О себе» в отдельный full screen экран.
- Кнопки редактирования в hero-профиле теперь открывают экран редактирования напрямую.
- На экране редактирования нет нижней навигации, чтобы поведение было ближе к открытой переписке в мессенджере.
- Добавлена верхняя панель с кнопкой назад и быстрым сохранением.
- Форма разделена на простые блоки: «Основное», «Детали», «Внешний вид».
- Инлайн-блок редактирования из вкладки «О себе» удалён.

## Зачем

Профиль остаётся витриной личности, а редактирование не раздувает основной экран и не ломает структуру вкладок.
