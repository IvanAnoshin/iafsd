# Community Moderation Implementation

## Что добавлено

Этот проход переводит модерацию сообществ из плана в рабочий слой.

### Посты сообщества

Добавлена модерация постов внутри сообщества:

- скрыть пост;
- вернуть пост;
- удалить пост мягким статусом `deleted`;
- закрепить пост;
- открепить пост;
- скрытые и удалённые посты не попадают в обычную ленту;
- модераторы видят скрытые/проверяемые посты внутри сообщества.

### Жалобы

Для постов добавлены поля состояния:

- `status`;
- `moderationReason`;
- `reportCount`;
- `hiddenAt`;
- `deletedAt`;
- `updatedAt`.

Жалоба на пост теперь увеличивает `reportCount`. После нескольких жалоб пост автоматически переводится в `under_review`.

Во вкладке `Модерация` отображается очередь новых жалоб по постам и комментариям сообщества.

### Комментарии

Добавлена модерация комментариев в очереди жалоб:

- скрыть комментарий;
- удалить комментарий;
- вернуть через backend-функцию при необходимости.

### Участники

Добавлены ограничения участников:

- мут на 24 часа;
- снятие мута;
- бан в сообществе;
- снятие бана через backend-функцию;
- забаненный пользователь не может повторно вступить или подать заявку.

В `CommunityMember` добавлены поля:

- `mutedUntil`;
- `mutedReason`;
- `bannedAt`;
- `banReason`.

### Права

Модерация доступна ролям:

- owner;
- admin;
- moderator.

Ограничения:

- модератор может модерировать только обычных участников;
- администратор не может модерировать владельца и другого администратора;
- владелец не может быть удалён/забанен через эти действия;
- пользователь не может модерировать сам себя.

### API

Добавлены/расширены маршруты:

- `GET /api/communities/[slug]/moderation` — журнал действий + очередь жалоб;
- `PATCH /api/communities/[slug]/posts/[postId]` — hide/restore/pin/unpin;
- `DELETE /api/communities/[slug]/posts/[postId]` — soft-delete поста;
- `PATCH /api/communities/[slug]/comments/[commentId]` — hide/restore/delete комментария;
- `PATCH /api/communities/[slug]/members/[memberId]` — роль или модерационное действие `mute/unmute/ban/unban`.

### UI

На странице сообщества добавлена вкладка `Модерация`.

В ленте сообщества для модераторов появились действия:

- закрепить/открепить;
- скрыть/вернуть;
- удалить.

В списке участников появились действия:

- мут 24ч;
- снять мут;
- бан.
