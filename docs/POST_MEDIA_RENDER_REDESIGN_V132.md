# POST MEDIA RENDER REDESIGN V132

## Что сделано
- Приведён контракт медиа поста к единому виду: `kind/type`, `url`, `thumbUrl`, `mime`, размеры, имя файла и служебные ключи.
- Исправлена причина, из-за которой прикреплённая картинка могла не отрисовываться в профиле: профильная карточка больше не ждёт `kind: photo`, а корректно понимает `image` и `video`.
- Чистые медиа-посты больше не получают технический текст `Медиа` в качестве видимого текста поста.
- В профиле добавлен полноценный медиа-блок поста:
  - одно фото/видео отображается крупно;
  - несколько вложений отображаются сеткой;
  - для видео показывается индикатор воспроизведения;
  - при битой ссылке остаётся аккуратная заглушка.
- В ленте добавлена нормализация медиа перед рендером, чтобы старые и новые форматы вложений отображались одинаково.
- Обновление поста через API теперь также не подставляет видимый текст `Медиа`.
