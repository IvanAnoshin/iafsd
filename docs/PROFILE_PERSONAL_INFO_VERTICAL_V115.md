# Profile personal info editor — v115

В этом проходе доработан full screen редактор профиля.

## Что изменилось

- Блок «Личная информация» переведён из двухколоночной сетки в вертикальный список.
- Поля свободного ввода стали многострочными и автоматически растягиваются вниз по мере набора текста.
- Длинный текст в полях больше не обрезается по горизонтали: он переносится на следующую строку.
- В выпадающих списках удалены дубли опции «Не указывать».
- Старые сохранённые значения сохраняются: если значение не входит в справочник, оно временно добавляется в список как текущий вариант.

## Почему так

На мобильной ширине двухколоночный вариант перегружал экран и обрезал текст. Вертикальный список проще читать, проще заполнять и лучше соответствует KISS-подходу проекта.
