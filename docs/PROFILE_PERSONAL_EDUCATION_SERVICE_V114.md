# Profile personal information fields v114

Добавлены поля личной информации в full screen редактор профиля:

- Школа
- Образование
- Военная служба

`Школа` и `Образование` остаются свободными текстовыми полями, потому что пользователь может указать конкретное учебное заведение, курс или направление.

`Военная служба` сделана выпадающим списком, потому что это поле предполагает ограниченный набор понятных вариантов. Значения сохраняются в `personalDetails` вместе с остальной личной информацией профиля.

Поля отображаются во вкладке `О себе` как в своём профиле, так и в публичном профиле другого пользователя, если приватность профиля не скрывает подробности.
