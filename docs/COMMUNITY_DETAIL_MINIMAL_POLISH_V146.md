# v146 — Community detail minimal polish

Что изменено:

- Фон страницы сообщества возвращён к общему Friendscape-градиенту вместо серого/белого поля вокруг мобильного контейнера.
- Верхняя строка стала компактной: назад к сообществам слева, управление — отдельной кнопкой справа для владельца/модератора.
- Большая неактивная кнопка `Вы владелец` убрана из hero-блока.
- Роль владельца/модератора теперь показывается компактной строкой с действием `Открыть управление`.
- Вкладки публичной страницы оставлены пользовательскими: `Лента`, `Медиа`, `Правила`, `Участники`.
- Админские разделы больше не торчат в публичном tabbar, но доступны через кнопку управления.
- Блок создания поста больше не раскрыт сразу на первом экране: вместо него компактная кнопка `Написать пост` / `Написать первый пост`.
- Composer раскрывается только по действию пользователя и сворачивается без сохранения черновика.
- Hero-блок стал ниже и спокойнее: компактнее обложка, аватар, мета и теги.

Логика API и модели не менялись.
