# COMMUNITIES CREATE PAGE V144

## Что изменено

- Создание сообщества вынесено из модального окна на отдельную страницу `/communities/create`.
- Кнопка `+` на `/communities` теперь открывает страницу создания, а не модалку.
- Форма оставлена простой: название, описание, тип доступа, теги.
- Адрес сообщества по-прежнему формируется backend автоматически из названия.
- Добавлен живой предпросмотр карточки сообщества в стиле Friendscape.
- Логика API создания не менялась: используется существующий `POST /api/communities`.

## Почему так

Модалка стала тесной для сценария создания. Отдельная страница лучше подходит для мобильного интерфейса: больше воздуха, понятнее контекст, меньше визуального шума.
