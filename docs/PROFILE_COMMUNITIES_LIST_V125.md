# PROFILE COMMUNITIES LIST V125

## Что сделано
- Кнопка статистики «Сообщества» в профиле теперь открывает отдельный список, как «Подписки», «Подписчики» и «Друзья».
- Экран списка использует тот же паттерн: карточка, кнопка «Назад», вертикальный список.
- Каждый пункт списка ведёт в карточку сообщества `/communities/[slug]`.
- Список показывает роль пользователя в сообществе и количество участников.
- Для скрытого списка остаётся приватное пустое состояние.
- Лимит загрузки сообществ профиля увеличен с 8 до 50, чтобы статистика и отдельный список не обрезались слишком рано.
- Правка применена к своему профилю и публичному профилю другого пользователя.

## Как создаётся сообщество сейчас
- Пользователь открывает раздел `/communities`.
- Нажимает кнопку «Создать».
- Заполняет форму: название, короткий адрес, описание, видимость, теги, цвет и первое правило.
- Клиент получает CSRF-токен и отправляет POST `/api/communities`.
- Сервер проверяет авторизацию, CSRF, rate-limit `community_create`, создаёт сообщество через `createCommunityForUser` и записывает audit log.
- После успешного создания пользователь переходит на `/communities/[slug]`, а создатель становится owner.
