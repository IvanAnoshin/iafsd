# V148 — убрать сообщества из пользовательских входов

Короткий проход по KISS: сообщества больше не показываются пользователю как основной сценарий.

Что изменено:
- добавлен флаг `COMMUNITIES_UI_ENABLED = false`;
- вкладка/настройка сообществ убраны из ленты;
- переход из поста в сообщество скрыт;
- репост в сообщество скрыт из sheet репоста;
- блоки, статистика и вкладки сообществ скрыты в личном и публичном профиле;
- загрузка сообществ для профилей отключена на UI-уровне.

Что не трогалось:
- backend API сообществ;
- Prisma-модели;
- маршруты `/communities`;
- хранилище и старые данные.

Следующий безопасный проход: закрыть прямой пользовательский вход в `/communities` редиректом или мягким экраном.
