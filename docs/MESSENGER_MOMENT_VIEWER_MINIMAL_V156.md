# V156 — минимальный просмотр моментов в мессенджере

Сделан короткий KISS-проход без сложной логики просмотра.

## Что изменено

- Добавлен минимальный viewer момента прямо внутри `/chat`.
- Клик по моменту в верхнем рейле теперь открывает viewer, а не ведёт на `/stories`.
- Viewer показывает автора, время/метку, заголовок, подпись и фото, если оно есть.
- Viewer закрывается по крестику, по клику на фон и по `Escape`.
- На время viewer блокируется фоновый скролл.
- Компонент `StoriesFoundationRail` получил необязательный `onSelect` для локального открытия моментов без роутинга.

## Что не трогалось

- Backend.
- Prisma.
- API `/api/stories`.
- Быстрые ответы на момент.
- Автопрогресс/свайпы между моментами.
