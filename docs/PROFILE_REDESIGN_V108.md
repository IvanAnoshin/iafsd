# Profile redesign v108

## Что изменено

- Верх профиля переведён в новый hero-формат: обложка, крупный аватар, имя, ник, описание и человеческие бейджи.
- Статистика стала компактной строкой: посты, друзья, подписчики, подписки.
- Основные действия вынесены в понятную нижнюю строку hero-блока.
- Разделы профиля разложены по вкладкам: посты, медиа, о себе, сообщества.
- Сообщества больше не занимают место до вкладок и не перегружают первый экран.
- Блок моментов убран из основного профиля, чтобы профиль не выглядел как набор случайных карточек.
- Похожий визуальный каркас применён и для публичного профиля другого пользователя.

## Принцип

Профиль теперь работает как чистая витрина человека: сначала личность и действия, потом контент по вкладкам.
