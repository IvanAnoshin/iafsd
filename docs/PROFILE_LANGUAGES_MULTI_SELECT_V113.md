# V113 — выбор языков и справочники личной информации

## Что изменено

- Поле «Языки» в full screen редакторе профиля теперь работает как мультивыбор: пользователь выбирает язык из выпадающего списка, выбранные языки отображаются чипами, каждый чип можно удалить.
- Ограничение на количество выбранных языков не ставится: пользователь может отметить столько языков, сколько знает.
- Поля с однозначным выбором переведены на выпадающие списки:
  - город сейчас;
  - родной город;
  - как обращаться;
  - статус отношений;
  - жизненная позиция.
- Свободный ввод оставлен только там, где он действительно нужен: занятие, проект/работа, образование, сайт и цитата.
- Личная информация хранится в `UserPublicProfile.personalDetails` как JSON, чтобы не плодить отдельные колонки под каждое необязательное поле.

## Приватность

Телефон и email не добавлялись. Публичное отображение личной информации подчиняется текущей видимости профиля.
