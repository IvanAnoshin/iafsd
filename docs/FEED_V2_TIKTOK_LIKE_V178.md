# Feed v2 — TikTok-like лента (v178)

Финальный проход серии v157–v178 перевёл `/feed` из классической карточной ленты в fullscreen/snap-механику просмотра.

## Что сделано

- Один пост занимает один экран или почти один экран.
- Вертикальный scroll-snap переключает публикации по одной.
- Текущая организация поста сохранена: автор, текст, медиа, реакции, комментарии, репост, меню.
- Лента разделена на режимы `Посты` и `Видео`.
- Источники ленты: `Друзья`, `Подписки`, `Новые люди`, `Все`.
- Порядок ленты: `Сначала новые` и `Рекомендации`.
- Комментарии открываются bottom sheet-ом поверх текущего поста.
- Репост/share открывается sheet-ом поверх текущего поста.
- Настройки ленты открываются отдельным компактным sheet-ом.
- При открытых sheet-ах snap-лента блокируется, а текущая позиция сохраняется.
- Причины показа, лишняя статистика и старые промо-блоки убраны с карточки.

## Финальная зачистка v178

- Удалён старый неиспользуемый page-level share/chat код, который дублировал работу `PostShareSheet`.
- Удалены неиспользуемые helper-функции старой ленты.
- Share/repost теперь остаётся в одном месте: `components/PostShareSheet.jsx`.

## Принципы реализации

- KISS: без новых таблиц Prisma и без сложного рекомендательного движка.
- Основная работа сделана на уровне `/feed` и CSS.
- Посты не перепридуманы с нуля — изменена механика просмотра и адаптация карточки.

## Проверить вручную

1. Открыть `/feed`.
2. Проверить вертикальное перелистывание постов.
3. Проверить `Посты / Видео`.
4. Проверить источники и порядок в настройках ленты.
5. Открыть комментарии и убедиться, что фон не свайпается.
6. Открыть share sheet и убедиться, что фон не свайпается.
7. Проверить фото, галерею, текстовый пост, ссылку, репост и видео.
8. Проверить мобильный экран с bottom-nav.
9. Запустить `npm run build` в локальном проекте с установленными зависимостями.

## Ограничения

- Полноценный build не прогонялся в рабочем окружении без `node_modules`.
- Рекомендации пока простые: активность + свежесть.
- Видео-режим использует существующие данные постов и медиа.
