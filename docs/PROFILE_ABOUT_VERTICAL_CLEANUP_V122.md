# PROFILE ABOUT VERTICAL CLEANUP V122

## Что сделано
- Убран лишний текстовый бейдж «Мой профиль» из hero-блока своего профиля.
- Убран бейдж «Личная страница».
- Убрана большая кнопка «Редактировать» из hero-блока, чтобы профиль не выглядел как панель управления.
- Быстрый вход в редактирование оставлен через компактную иконку карандаша в hero-блоке.
- Во вкладке «О себе» информация теперь отображается вертикальным списком, а не сеткой в две колонки.
- Пустые поля больше не выводятся карточками с текстом «Не указано». В блок попадают только заполненные значения и служебно полезный адрес профиля.
- Аналогичное вертикальное отображение добавлено для публичного профиля другого пользователя.
