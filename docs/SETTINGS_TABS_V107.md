# Settings tabs v107

Проход v107 упорядочивает экран `/settings` без изменения бизнес-логики.

## Что изменено

- Добавлена горизонтальная вкладочная навигация внутри настроек.
- Разделы сгруппированы по вкладкам:
  - `Аккаунт`: сессии, устройства, экспорт, удаление и выход.
  - `Приватность`: видимость профиля, сообщений и уведомления.
  - `Защита`: защищённые чаты, пароль, секретный ответ, passkey, recovery-фраза и резервные коды.
  - `Вид`: тема, режимы зрения и уменьшение анимаций.
  - `Поддержка`: обращения и отзыв о бете.
  - `Админ`: административные панели, только если пользователь администратор.
- Секции не размонтируются сложной логикой, а скрываются простым CSS-классом, чтобы не ломать существующие формы и обработчики.

## Принцип реализации

KISS: один маленький компонент `SettingsTabs`, один state `activeSettingsTab`, один helper `tabClass` и CSS для скрытия неактивных секций.
