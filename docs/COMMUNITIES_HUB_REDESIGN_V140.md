# COMMUNITIES_HUB_REDESIGN_V140

Переделан главный экран `/communities` как пользовательская витрина, без смешивания с админкой.

Что изменено:
- нижний слой API не тронут;
- главный экран стал отдельным hub-экраном с шапкой, поиском и фильтрами;
- добавлены фильтры: «Для тебя», «Популярные», «Новые», «Мои»;
- invite-код вынесен в отдельную компактную модалку;
- создание сообщества осталось минимальным и вынесено в отдельную модалку;
- карточки сообществ перерисованы: обложка, тип доступа, участники, теги, статус пользователя;
- «Мои сообщества» отображаются отдельной быстрой горизонтальной полосой;
- админские сценарии не показываются на главной витрине.

Идея прохода: `/communities` должен ощущаться как место для поиска и входа в сообщества, а не как панель управления.
