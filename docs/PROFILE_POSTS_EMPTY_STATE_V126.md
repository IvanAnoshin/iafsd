# PROFILE POSTS EMPTY STATE V126

## Что сделано
- Заменён скучный блок «Пока нет постов.» на полноценный визуальный empty-state.
- На своём профиле теперь показывается аккуратная карточка с мини-иллюстрацией, бейджем, заголовком и коротким пояснением.
- В публичном профиле другого пользователя такой же пустой блок используется для состояния без публикаций.
- Кнопку создания публикации не возвращал, чтобы не ломать ранее очищенный профиль.
